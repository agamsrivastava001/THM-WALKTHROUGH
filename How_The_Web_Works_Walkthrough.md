# 🌐 TryHackMe: How The Web Works Walkthrough

## 🎯 Objective
Understand the foundational infrastructure of the World Wide Web, including how browsers communicate with servers, the structure of web requests, and core protocols.

---

## 🧭 Core Concepts Covered

### 1. Web Suite & HTTP/S Protocols
- **What I Learned:** Understood the difference between HTTP and HTTPS (SSL/TLS encryption). 
- **Key Takeaway:** How data travels securely from a client to a server without being intercepted (MitM protection).

### 2. Request and Response Architecture
- **Structure:** Analyzed HTTP Request Methods (`GET`, `POST`, `PUT`, `DELETE`) and Headers.
- **Status Codes:** Learned what different server responses mean:
  - `200 OK` (Success)
  - `301/302` (Redirection)
  - `403 Forbidden` / `404 Not Found` (Client Errors)
  - `500 Internal Server Error` (Server Error)

### 3. DNS (Domain Name System)
- **Concept:** The phonebook of the internet. Learned how a domain name (like google.com) resolves into an IP address using Root, TLD, and Authoritative servers.
